# Proyecto Semestral TIA - Avance 2

Proyecto: Comparación de SARSA, Q-Learning y Monte Carlo en un gemelo digital simplificado de un almacén inteligente.

Este repositorio contiene el notebook, dataset e informe correspondientes al Avance 2 del proyecto semestral de Taller de Inteligencia Artificial.

## Archivos principales

- `exploracion_smart_warehouse_avance2.ipynb`: notebook con exploración del dataset, preprocesamiento, implementación de Q-Learning y resultados iniciales.
- `dataset/`: archivos CSV y JSON utilizados para representar el almacén inteligente.

## Instrucciones de ejecución

1. Abrir el notebook en Jupyter Notebook, Google Colab o VS Code.
2. Mantener los archivos del dataset en la carpeta `dataset`.
3. Ejecutar las celdas en orden.
